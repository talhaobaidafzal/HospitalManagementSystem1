# Hospital Management System (OOP in Java)

This is a console-based Hospital Management System built using Object-Oriented Programming concepts in Java. It allows management of doctors and patients with features like adding, listing, and assigning doctors.

## 💡 Key Features
- Add new doctors and patients
- Assign doctors to patients
- List all doctors and patients
- Input validation with exception handling

## 📚 Technologies Used
- Java (OOP concepts: Inheritance, Polymorphism, Encapsulation, Abstraction)
- Java Collections Framework
- Exception Handling

## 📁 File Structure
- `Person.java` – Abstract base class for Doctor and Patient
- `Doctor.java` – Inherits from Person
- `Patient.java` – Inherits from Person
- `Hospital.java` – Handles core operations
- `Main.java` – Entry point of the application
